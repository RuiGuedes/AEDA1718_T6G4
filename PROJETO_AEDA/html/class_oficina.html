void Oficina::buyPiece ( )

Compra uma peca para reparar uma bicicleta avariada. Pede ao utlizador para escolher uma bicicleta para reparar, da lista apresentada, escolher a peca que pretende reparar, da lista apresentada, escolher o fornecedor, e ainda e pedido para inserir o preco da peca que pretende comprar.
